The UI uses **Glassmorphism** to indicate informational density.

*   **Annamaya (Physical):** Opaque, solid borders (`border-2`), dark surfaces (`bg-slate-950`).
*   **Pranamaya (Vital):** Animated pulses, glowing connector lines (`stroke-width: 2px`).
*   **Manomaya (Mental):** Semi-transparent panels (`backdrop-blur-md`), readable typography (Rubik/Inter).
*   **Vijnanamaya (Wisdom):** 3D objects, floating nodes, wireframe structures.
*   **Anandamaya (Bliss):** Minimalist white-space, high-key lighting, glassmorphism (`backdrop-blur-xl`).
