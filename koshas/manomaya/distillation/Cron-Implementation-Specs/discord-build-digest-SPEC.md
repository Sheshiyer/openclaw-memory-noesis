### **5. Source File Scanning Logic**

**What Constitutes a "Build"?**

**Primary Sources (scan last 7 days):**

1. **Git Commits** (if repo available)
   - Command: `git log --since="7 days ago" --oneline --all`
   - Pattern: Look for completion markers ("feat:", "fix:", "refactor:")
   - Failure markers: "revert:", "WIP:", "TODO:"

2. **Memory Logs** (`koshas/manomaya/logs/YYYY-MM-DD.md` for last 7 days)
   - Pattern: Search for:
     - "Built", "Shipped", "Completed"
     - "Failed", "Stalled", "Abandoned"
     - "Learning:", "Insight:", "Realized"
   - Aggregation: Group by day, identify themes

3. **Nightly Builds** (`koshas/manomaya/meta/nightly-builds.md` or similar)
   - Pattern: Check "Status: âœ… Done" vs "Status: â³ In Progress" vs "Status: âŒ Blocked"
   - Trend: Are more items completing or stalling?

4. **PARA Project Folders** (`01-Projects/`)
   - Command: `find 01-Projects -type f -mtime -7` (files modified in last 7 days)
   - Pattern: Changed files = active work
   - Exclusions: Ignore `.DS_Store`, temporary files

5. **Cron Job Logs** (`_System/cron/logs/` if available)
   - Pattern: New jobs added, old jobs modified, execution errors
   - Signal: System evolution, infrastructure work

**Failure Extraction Protocol:**

**1. Git Reverts**
- Command: `git log --since="7 days ago" --grep="revert" --oneline`
- Context: What was reverted? Why? (Check commit message or linked issue)

**2. Error Logs**
- Files: `koshas/manomaya/logs/YYYY-MM-DD.md`, search for "ERROR", "FAILED", "Exception"
- Pattern: Same error recurring = systemic issue (prioritize in digest)

**3. Abandoned Branches**
- Command: `git branch --no-merged main | xargs -n1 git log -1 --since="7 days ago" --format="%cr %s"`
- Signal: Branches created but not merged = exploration that didn't pan out (valid learning)

**4. Stalled Tasks**
- Source: `koshas/manomaya/meta/nightly-builds.md` or daily logs
- Pattern: Task marked "â³ In Progress" for 3+ consecutive days without update = stall
- Question: Why stalled? Confusion? Missing input? Lost interest?

**5. Explicit Failure Notes**
- Source: Daily logs, search for:
  - "This didn't work"
  - "Realized this approach won't work"
  - "Abandoning"
  - "Need to rethink"

**Aggregation Algorithm:**
1. Scan all sources (5-10 minutes of file reading)
2. Extract 10-15 raw items (mix of builds, failures, learnings)
3. Buddhi discerns: Which 5-7 are most valuable for community?
4. Group into: Shipped, Stalled, Failed, Learned
5. Chitta synthesizes: What's the week's pattern?
