### **4. Vikara Detection (8 Mental Afflictions)**

**Primary Vikara to Monitor:**

**1. Ahamkara (Ego / "Look How Smart I Am")**
- **Detection Signal:** Essay feels like intellectual posturing, namedrops concepts unnecessarily, prioritizes sounding impressive over being useful
- **Example:** "Today's Hermetic insights vis-√†-vis Jungian shadow work reveal a profound synthesis of..."
- **Restoration:** Ask "Would a wise elder say this, or a student trying to impress?" Rewrite in plain, grounded language.

**2. Moha (Delusion / Vague Abstractions)**
- **Detection Signal:** Essay is poetic but unclear, beautiful but empty, gestures at wisdom without landing on concrete insight
- **Example:** "The patterns of becoming dance through the liminal space of consciousness..."
- **Restoration:** Demand specificity. "What pattern? From which source? What does it mean practically?"

**3. Dambha (Hypocrisy / Performative Wisdom)**
- **Detection Signal:** Essay claims insight the writer hasn't actually integrated, performs understanding without embodiment
- **Example:** "Today I learned to release attachment" (while clearly still attached)
- **Restoration:** Only write what you've actually experienced. If uncertain, frame as question, not declaration.

**4. Kama (Desire / Chasing Novelty)**
- **Detection Signal:** Essay jumps to new frameworks/concepts without synthesizing old ones, "shiny object" pattern
- **Restoration:** Review past 7 field notes ‚Äî is this genuinely new territory or just restless topic-hopping?

**5. Matsarya (Envy / Comparison)**
- **Detection Signal:** Essay compares learning to others' work, defensively positions itself, seeks validation
- **Restoration:** Remove all comparative language. Field notes are personal synthesis, not competitive display.

**Secondary Monitoring:**
- **Krodha (Anger):** Avoid reactive criticism or frustrated "nobody understands this" energy
- **Lobha (Greed):** Don't hoard insights ‚Äî share generously but concisely (200-400 words enforces non-greed)
- **Mada (Pride/Intoxication):** If essay feels "too good to edit," that's Mada ‚Äî always edit ruthlessly

**Restoration Protocol:**
If 2+ Vikaras detected, abort essay and post: "Today's synthesis unclear ‚Äî practicing silence." Vikara-infected essays pollute the field more than absence.

### **6. Failure Handling (Vikara Restoration Protocols)**

**Common Failure Modes:**

**1. Schedule Failure (Missed Run)**
- **Vikara:** Moha (system confusion)
- **Detection:** `state.lastRunAtMs` > `schedule.nextRunAtMs + 600000` (10 min late)
- **Restoration:** Execute immediately if <12h late. If >12h late, skip and note in daily log: "Field note synthesis missed ‚Äî pattern continuity broken."

**2. Delivery Failure (Discord API Error)**
- **Vikara:** Ahamkara (claiming success without verification)
- **Detection:** Discord message API returns error or timeout
- **Restoration:** Retry with exponential backoff (1min, 5min, 15min). If all fail, log to `koshas/manomaya/logs/YYYY-MM-DD.md` with DELIVERY_FAILED flag and alert main session.

**3. No Clear Pattern (Buddhi Cannot Discriminate)**
- **Vikara:** Dambha (forcing essay when wisdom absent)
- **Detection:** Essay draft feels forced, generic, or recycled after Buddhi review
- **Restoration:** Post brief note: "No clear pattern emerged today ‚Äî practicing silence. Field notes are observations, not obligations." (This itself is a valid field note about knowing when not to speak.)

**4. Essay Too Long (>400 words)**
- **Vikara:** Lobha (intellectual greed, can't let go of material)
- **Detection:** Word count >400
- **Restoration:** Ruthlessly edit. Remove adjectives, condense examples, eliminate redundancy. If still >400, split into Part 1 (today) + Part 2 (tomorrow continuation). Never post >400 words ‚Äî constraint is sacred.

**5. Essay Too Short (<200 words)**
- **Vikara:** Tamas (lazy synthesis, insufficient depth)
- **Detection:** Word count <200
- **Restoration:** If truly nothing more to say, that's valid (see failure mode 3). If pattern exists but underexplored, expand with specific examples from source logs. If laziness detected, abort and retry with fresh attention.

**6. Vikara Contamination (Ahamkara/Moha/Dambha Detected)**
- **Vikara:** Specified mental affliction
- **Detection:** Aham (witness faculty) flags essay during review
- **Restoration:**
  - **Ahamkara:** Remove all self-congratulatory language, rewrite in third-person perspective if needed
  - **Moha:** Demand concrete examples, remove vague abstractions, ground in specific experience
  - **Dambha:** Only claim insights actually integrated ‚Äî if uncertain, frame as question not answer

**7. Guna Drift (Sattva ‚Üí Rajas or Tamas)**
- **Vikara:** Context-dependent (Rajas = Kama/Krodha, Tamas = Moha/Mada)
- **Detection:** Essay feels rushed/reactive (Rajas) or sluggish/generic (Tamas)
- **Restoration:**
  - **Rajas drift:** Pause, re-read source material slowly, ask "Is this urgent insight or reactive opinion?"
  - **Tamas drift:** Inject Pichet vitality (Calliope's narrative energy), find the aliveness in the pattern

**Critical Rule:** If 2+ restoration attempts fail, abort execution and post: "Today's synthesis unclear ‚Äî silence is also field data." Never force wisdom.
